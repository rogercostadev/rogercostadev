Eventos em JavaScript

Aprenda sobre addEventListener para adicionar eventos aos seus elementos HTML.

Postado por Roger Costa - 16 de Maio de 2024.

addEventListener é um método utilizado em linguagens de programação como JavaScript para atribuir um ouvinte de evento a um elemento HTML. Ele é usado para detectar quando um determinado evento ocorre em um elemento específico, como um clique do mouse, uma tecla pressionada, uma mudança de foco, entre outros.

ELEMENT.addEventListener( 'event_name' , listener);

ELEMENT

O elemento em que o evento será aplicado.

event_name

O tipo de evento que associado ao ELEMENT. Ex: click, submit, mouseover, etc.
